&#x20;Product Web Scraper



&#x20;📌 Overview

This project is a Python-based web scraper that extracts product data from an e-commerce website and saves it into a structured Excel file.



🚀 Features

\- Scrapes product title, price, rating, availability

\- Extracts product links

\- Converts rating into numeric values (1–5)

\- Saves clean data into Excel format



🛠️ Tech Stack

\- Python

\- requests

\- BeautifulSoup

\- pandas

\- openpyxl



▶️ How to Run



1\. Clone the repository:

git clone https://github.com/afthabsadique602/product-web-scraper
.git



2\. Install dependencies:

pip install -r requirements.txt



3\. Run the script:

python scraper.py
